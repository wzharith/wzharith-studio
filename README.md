# WZHarith Music

**Professional Saxophone Performance for Weddings & Events**

This repository contains the complete business setup for WZHarith Music, a saxophone performance enterprise based in Malaysia.

## Repository Structure

```
saxophone/
├── Archive/                    # Past wedding performance files (backing tracks)
├── Master List/                # Master collection of all backing tracks
├── docs/                       # Business documentation
│   ├── business_setup.md       # SSM registration checklist
│   ├── pricing_packages.md     # Service packages and pricing
│   └── grant_applications.md   # Grant opportunities guide
├── stats/                      # Analytics tools
│   ├── song_analytics.py       # Python script for song statistics
│   ├── performance_report.txt  # Generated statistics report
│   └── song_statistics.csv     # Song data in CSV format
└── platform/                   # Website source code (Next.js)
```

## Getting Started

### Song Analytics

Run the analytics script to generate fresh statistics:

```bash
python3 stats/song_analytics.py
```

### Website (Development)

```bash
cd platform
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
